Project     : OneIDNet
Version     : 1.0
Duration    : 13 Feb 2015 - 27 Feb 2015

Details
=======
1) login page functionality with session management and cookie.
2) Registration page functionality without auto detect, mobile extension.
3) forgot password via email without sending email functionality.
4) forgot password via mobile without sending sms functionality.
5) logout functionality without updating logout time.

Libraries:
===========
1) GetCST_Info[Tested OK]:[Open source]Library to return state,country,city,region etc information
2) GetIP_loc[Tested OK]:[Commercial]Library to return state,country,city,region etc information
3) GetUnique_Domains[Not working for https]: Library to extract unique URLs.
4) Mobile_Detect[Tested OK]: Library to return the device type i.e., Computer, Tablet or Mobile.
5) Tag_Filters[Tested OK]: Library gives unique hash tags 
6) Uuid[Tested OK]: Library which gives the unique alphanumeric 37 character long id.
